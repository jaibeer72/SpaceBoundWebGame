## Website branch rules

### Nameing Features 

* lowecase every branch name
* format [feature-name/attribute]
    * example 
```git
image-card/masking-image
``` 

### Nameing Tests 

* lowecase every branch name
* format [test/feature/attribute]
    * example 
```git
test/image-card/masking-image-position
```
